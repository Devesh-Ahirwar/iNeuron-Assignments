3. What modes do File objects for reader and writer objects need to be opened in?
The file objects for reader and writer objects need to be opened in either "r" (read) mode or "w" (write) mode. "r" is used for reading an existing file and "w" for creating a new file or overwriting an existing file.

5. What do the keyword arguments delimiter and line terminator do?
The delimiter and line terminator keyword arguments specify the character used to separate values in a row and the character used to separate rows, respectively. For example, the default delimiter is "," and the default line terminator is "\r\n".
